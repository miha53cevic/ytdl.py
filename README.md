# Youtube Downloader
A minimal ytdl wrapper that downloads youtube videos to mp3

## Usage
```bash
    ytdl ${url}
``` 
At the end press enter to exit

## Install
```bash
    cd youtubeDL.py
    pip3 install .
```
You also need ffmpeg installed
https://ffmpeg.org/
